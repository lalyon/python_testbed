# python_testbed
Bit- and Byte-wise file operations.

The `BinaryFile` class in `src/python_testbed/binary_ops.py` contains the following methods:

- `readFileAsBytes()`
- `writeToFile()`
- `totalBytes()`
- `zeroByte()`
- `bitSwap()`


Run the tests from the parent project directory

```python
>>> cd python_testbed
>>> pytest
```
